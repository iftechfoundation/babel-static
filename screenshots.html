See the Treaty of Babel in Action

The Treaty of Babel allows any compliant tool to access bibliographic and cover data for Treaty-enhanced games. Check out these screenshots of Babel-enhanced software.

  • IFDB can provide XML iFiction documents for a given IFID. See the IFDB API documentation for more information.
    For example: https://ifdb.org/viewgame?ifiction&ifid=7B5A779B-4653-43DB-A516-F475DDC12987
  • Windows Frotz uses Treaty features to display cover art and descriptions:Bureaucracy, Curses, Jigsaw
  • Spatterlight provides a story browser to organize and search your collection: Trinity
